# 🤖 AI Chatbot using Streamlit and Google Gemini

This project is a simple AI chatbot built using **Streamlit** and **Google Gemini API**.
It allows users to interact with an AI assistant through a clean chat interface.

## 🚀 Features

* Interactive chat interface
* Built with Streamlit
* Uses Google Gemini generative AI model
* Maintains chat history during the session
* Error handling for API failures
* Lightweight and easy to deploy

## 🛠️ Technologies Used

* Python
* Streamlit
* Google Generative AI (Gemini API)

## 📂 Project Structure

```
ai_ds_chatbot/
│
├── ChatBot.py          # Main chatbot application
├── requirements.txt    # Project dependencies
└── README.md           # Project documentation
```

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/yourusername/ai_ds_chatbot.git
```

2. Navigate to the project folder

```
cd ai_ds_chatbot
```

3. Install dependencies

```
pip install -r requirements.txt
```

## 🔑 API Key Setup

Create a Gemini API key from:

https://aistudio.google.com/app/apikey

For **Streamlit Cloud deployment**, add the key in **Secrets**:

```
API_KEY = "your_api_key_here"
```

## ▶️ Running the Application

Run the Streamlit app using:

```
streamlit run ChatBot.py
```

The chatbot will open in your browser.
